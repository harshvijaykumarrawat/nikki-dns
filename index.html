NIKKI DNS

Node.js adblock/malware domain blocker with SQLite3 support

Features

  • Fetches blocklists from multiple CDNs
  • Supports large SQLite3 databases
  • Runs as a systemd service on Debian/Ubuntu
  • Automatically updates blocklists
  • Logs to syslog

Installation

wget https://github.com/harshvijaykumarrawat/nikki-dns/releases/download/v1.0.0/nikki-dns.deb
sudo dpkg -i nikki-dns.deb
sudo systemctl status nikki-dns

Development

git clone https://github.com/harshvijaykumarrawat/nikki-dns.git
cd nikki-dns
npm install
node index.js
